### First stage
  * Clone this repository
      `git clone https://github.com/imhicihu/Biblio-search.git`
  * Install Electron Packager (if first time)
    `npm install electron-packager -g`
  * Go into the repository
    `cd Biblio-search`
  * Install dependencies 
    `npm install`
  * Run the app
    `npm start`
  * Building the Executable App
    `cd Biblio-search`
    `npm run package-win`
    `npm run package-mac`
